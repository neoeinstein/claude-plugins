# askama Plugin

Last verified: 2026-02-04

## Purpose

Guidance for writing and debugging Askama templates (Jinja2-like templates compiled to Rust). Two skills migrated from personal dotfiles.

## Contracts

- **Exposes**: `askama:askama-rust-templates` skill, `askama:askama-debugging` skill
- **Guarantees**: Skills are standalone, no hooks or external dependencies
- **Expects**: User working with Askama crate in a Rust project

## Current State

v0.1.0 stub. Skills migrated verbatim. See `docs/design-notes.md` for planned improvements (anti-rationalization tables, HTMX integration, reference doc split).
