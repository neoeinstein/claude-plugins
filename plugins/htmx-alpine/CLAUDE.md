# htmx-alpine Plugin

Last verified: 2026-02-04

## Purpose

Patterns for building server-driven web applications with HTMX and Alpine.js. Language-agnostic guidance focused on the interaction model.

## Contracts

- **Exposes**: `htmx-alpine:htmx-alpine` skill
- **Guarantees**: Skill is standalone, language-agnostic, no hooks
- **Expects**: User working with HTMX and/or Alpine.js in any server-side framework

## Current State

v0.1.0 stub. Seeded from patterns used in constant-gathering project. See `docs/design-notes.md` for planned scope (SSE/WS, Alpine stores, framework-specific variants).
